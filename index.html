ℹ️ Información

  • URL Individual: Descarga un solo video o audio desde su URL directa de YouTube.
  • Archivo .txt: Sube un archivo de texto con múltiples URLs o nombres de canciones junto al nombre del artista (una por línea) para descarga masiva. También puedes escribirlos directamente sin subir ningún archivo.
  • Playlist: Descarga todos los videos de una playlist de Spotify u otras plataformas compatibles. Necesitas usar Exportify o hacer el archivo .csv tu mismo.

❓ Preguntas Frecuentes

URL Individual

  • ¿De dónde saco la URL?: Abre YouTube, busca la canción que quieras, copia la URL que aparece arriba (caso de PC) o copia la URL de compartir video.
  • ¿Cómo copio y pego la URL?: Usa Ctrl + C para copiar y Ctrl + V para pegar.

❓ Preguntas Frecuentes

Archivos .txt / Escritura directa

  • ¿Qué estructura o formato deben de tener las canciones?: Escribe el nombre de la canción y del artista en la misma línea, algo así como: "i dont care violent vira". Queda así:
    i dont care violent vira
    damn girl rock n chile
    piel con piel la santa grifa
  • ¿Puedo mezclar URLs y nombres?: Sí, cada línea puede ser una URL de YouTube o un nombre de canción.

❓ Preguntas Frecuentes

Playlist / Excel (.csv)

  • Le subo mi archivo Excel y no se descarga: Revisa que estes subiendo el tipo de archivo adecuado, debe de ser .csv, actualmente no soportamos otros formatos como xlsx.
  • Con que plataformas funciona?: Actualmente solo se puede para playlist de Spotify

📚 Cómo usar Exportify

  1. Abre Exportify
  2. Haz click en "Get Started"
  3. Inicia sesión en Spotify
  4. Elige la playlist y dale click en "Export"
  5. Se descargará un .csv — pásalo a .xlsx: Archivo → Guardar como → Libro de Excel
  6. Arrastra el .xlsx al área de drop y descarga.
