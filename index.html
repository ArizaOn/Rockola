ℹ️ Información

  • URL Individual: Descarga un solo video o audio desde su URL directa de YouTube.
  • Archivo .txt: Sube un archivo de texto con múltiples URLs o nombres de canciones junto al nombre del artista (una por línea) para descarga masiva.
  • Playlist: Descarga todos los videos de una playlist de Spotify u otras plataformas compatibles.

❓ Preguntas Frecuentes

URL Individual

  • ¿De dónde saco la URL?: Abre YouTube, busca la canción que quieras, copia la URL que aparece arriba (caso de PC) o copia la URL de compartir video.
  • ¿Cómo copio y pego la URL?: Usa Ctrl + C para copiar y Ctrl + V para pegar.

❓ Preguntas Frecuentes

Archivos .txt

  • ¿Qué estructura o formato deben de tener las canciones?: Debes escribir nombre de la canción y del artista, algo así como: "i dont care violent vira", no es necesario separar nombre de canción y artista, para la siguiente canción solo separa por un salto de línea. Te va a quedar como:
    i dont care violent vira
    damn girl rock n chile
    piel con piel la santa grifa

❓ Preguntas Frecuentes

Playlist / Excel (.xlsx)

  • Le subo mi archivo Excel y no se descarga: A veces pasa, vuelve a intentarlo, sino checa si el archivo que le estas metiendo es .xlsx o formato de libro. Si lo descargaste de Exportify abre el archivo excel → Archivo → Guardar como → Examinar → Tipo → Libro (es la opción de hasta arriba).
  • Le subo la URL de mi playlist de Spotify pero no se descarga: Lo sentimos, aún no tenemos esa función :( , pero puedes donar para que nos animemos a programarla :D .

📚 Cómo usar Exportify

Lo que queremos es obtener un archivo xlsx de Excel que represente las canciones de algún álbum o playlist que hayas guardado en Spotify.

  1. Abre Exportify
  2. Haz click en "Get Started"
  3. Inicia sesión en Spotify
  4. Elije la playlist que vas a querer descargar, ya que sepas cuál descargarás dale click en "Export"
  5. Se te descargará un archivo Excel .csv (este formato no nos sirve)
  6. Pásalo a formato xlsx: Abre el archivo .csv que se te descargó → Archivo → Guardar como → Examinar → Tipo → Libro de Excel (este es el tipo de archivo .xlsx)
  7. Arrastra el archivo excel .xlsx hacia el área de drop y descarga.
